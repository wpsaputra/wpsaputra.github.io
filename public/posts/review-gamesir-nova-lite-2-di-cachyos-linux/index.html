Review Gamesir Nova Lite 2 Di Cachyos Linux

Review kompatibilitas Gamesir Nova Lite 2 di Linux

November 24, 2025

Setelah muak dengan arah perkembangan Windows 11 dan fitur-fiturnya yang semakin tidak ramah terhadap old hadware saya bermigrasi ke salah satu distro linux yaitu cachy os. Setelah sebelumnya menggunakan debian untuk vps, saya sangat mengapresiasi ketersediaan driver dan setup yang mudah dari cachy os khususnya untuk gaming dan kemudahan penggunaan di desktop saya. Steam dan Nvidia berjalan dengan sangat baik setelah proses instalasi.

Dalam bermain game tentunya kurang lengkap tanpa joystik terutama untuk game hack and slash seperti Devil May Cry, Ninja Gaiden, Metal Gear Rising Revengeance dsb. Gamesir nova lite 2 sendiri secara resmi menawarkan kompatibilitas untuk Windows, Android, iOS, Switch dengan koneksi yang lengkap via bluetooth, wireless receiver usb, atau menggunakan kabel yang ada di paket penjualan.

Untuk ujicoba di cachyos saya menggunakan steam dengan menggunakan receiver connection. Joystik langsung terhubung dan dapat dibaca oleh steam sebagai controller PS2. Semua fungsi controller terbaca dengan baik tetapi fungsi getar dari joystik tidak terdetect pada mode default. Tetapi jangan khawatir dengan menggunakan kombinasi tombol start dan select yang ditekan bersamaan akhirnya mode controller bisa berubah ke mode xbox dan fungsi getar bisa berjalan dengan baik.

Impresi yang saya dapat selama penggunaan joystik ini adalah dengan harga yang cukup murah Rp. 200 ribuan kita bisa mendapatkan joystik dengan kompatibilitas os yang sangat baik, multi koneksi via bluetooth, wireless, dan kabel. Build quality yang sangat baik, nyaman digenggam. Sangat worthed it untuk controller di harga 200 ribuan jika kita tidak membutuhkan fitur gyroscope. Berikut link shopee yang saya gunakan jika tertarik untuk membeli.
